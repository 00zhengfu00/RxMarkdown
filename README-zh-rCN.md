# RxMarkdown

RxMarkdown 是一个运用 RxJava API 来实现在   `android.widget.TextView` 或 `android.widget.EditText` 中编辑和显示基础 markdown 语法的 Android 库。 

## 支持语法 

- [x] 标题 `# ` / `## ` / `### `
- [x] 区域引用 `> `
- [x] 嵌套区域引用 `> > `
- [x] 粗体 `**`
- [x] 斜体 `*`
- [x] 粗体和斜体嵌套
- [x] 有序列表 `1. `
- [ ] 有序列表
- [x] 无序列表 `* ` /  `+ ` / `- `
- [x] 嵌套无序列表
- [x] 图片 `![]()`
- [x] 链接 `[]()`
- [x] 行内代码 `` `
- [x] 代码区块 ` ``` `
- [x] 反斜杠 `\`
- [x] 分割线 `***` / `*****` / `---` / `-----------------`
- [x] 删除线 `~~`
- [x] 注脚 `[^]`
- [x] Todo `- [ ] ` / `- [x]`

### 其他语法

- [x] 居中 `[]`

- [ ] 文字下划线 `待定`
